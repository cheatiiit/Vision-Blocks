VisionBlocks is a platform that makes computer vision programming much easier via drag-and-drop user interface.

Dev version is constantly updated on http://vblocks.media.mit.edu/. Please use latest Google Chrome to enjoy the cutting-edge web technologies. There is an older version located at http://visionblocks.org/.

Brought to you by Camera Culture Group at the MIT Media Lab <http://cameraculture.media.mit.edu>.

Feedbacks and collaboration welcomed.
